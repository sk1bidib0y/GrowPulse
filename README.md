# GrowPulse: Prototype

GrowPulse is an app that helps farmers and gardening enthusiasts automate their greenhouses or gardens, with the correct setup with hardware, GrowPulse can do the following:
  
- [x] Keep track of Plant Telemetry such as,

	1. Crop Type
	2. Temperature Data
	3. Humidity (%)
	4. Growth Rate
	5. Soil Moisture
	6. When sprinklers are active

- [x] Allow users to view data mentioned above with statistical graphs

	1. Box Plot
	2. Line Graph
	3. Scatter Plot
	4. Bar Chart
![A box plot showing Growth Rate of Crop Types grouped by Crop types](ReadmeContent\Stats_data.png)
---
- [x] Allow users to export data by limiting it to set specific fields.
- [x] Keep track of real time metrics such as,

	1. Temperature
	2. Humidity Percentage
	3. Soil Moisture Percentage
	4. Sprinkler Status (On / Off)
![Dashboard showing real-time metrics](ReadmeContent\rt_metrics.png)
